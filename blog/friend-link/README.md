# Help

该页面为我的博客`sucrayon.top`/`sucrayon.github.io`的友链添加页面，大家可以自行根据规则添加友情链接

字段说明

| 字段名（field） | 描述（description） |
| --------------- | ------------------- |
| name            | 主页名称            |
| link            | 链接                |
| avatar          | 显示的头像          |
| descr           | 友链描述            |
| order           | 友链顺序            |



添加友链指引

1.  在`assets`下按照字段说明新增一个`json`文件
2.  提交PR
3.  等待合入，合入即添加成功
