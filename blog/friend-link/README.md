# Help

该页面为我的博客`sucrayon.top`的友链添加页面，大家可以自行根据规则添加友情链接

`friend-link.json`

字段说明

| 字段名（field） | 描述（description） |
| --------------- | ------------------- |
| name            | 主页名称            |
| link            | 链接                |
| avatar          | 显示的头像          |
| descr           | 友链描述            |



添加友链指引

1.  在`link_list`中按照字段说明追加一个对象
2.  提交MR
3.  等待合入，合入即添加成功
